# 📰 News Aggregator Web App

## 📌 Overview

This project is a **full-stack News Aggregator web application** that collects and displays the latest news from global sources in one place.
It provides users with a simple, responsive interface to browse, search, and filter news articles in real time.

The application uses a **frontend (HTML, CSS, JavaScript)** and a **backend (Node.js, Express)**, with data fetched from an external news API.

---

## 📊 Dataset

* Data is fetched dynamically from **NewsAPI**
* Includes:

  * News title
  * Description
  * Image
  * Article URL
* No static dataset is stored; data is updated in real time via API calls

## ⚙️ Project Workflow / Steps

1. Designed UI layout using HTML and CSS
2. Built responsive card-based layout for news display
3. Created backend server using Node.js and Express
4. Integrated NewsAPI to fetch real-time news
5. Developed REST API endpoint (`/news`)
6. Connected frontend to backend using `fetch()`
7. Implemented:

   * Search functionality
   * Category filtering (e.g., Tech, Sports)
8. Handled errors (CORS, API failures, HTTP errors)
9. Deployed:

   * Backend on Render
   * Frontend on Netlify

## ▶️ How to Run the Project

### 🔹 1. Clone Repository

```bash

cd news-aggregator ----> https://news-h.netlify.app/
```

---

### 🔹 2. Install Backend Dependencies

```bash
npm install
```

---

### 🔹 3. Create `.env` file

```env
API_KEY=your_newsapi_key
```

---

### 🔹 4. Run Backend Server

```bash
node server.js
```

---

### 🔹 5. Open Frontend

* Open `index.html` using Live Server
  OR
* Visit deployed site (Netlify link)
